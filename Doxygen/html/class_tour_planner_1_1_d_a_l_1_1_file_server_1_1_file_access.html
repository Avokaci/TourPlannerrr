Detailed Description

FileAccess class, which is responsible for all the interaction with the filesystem. The purpose of this class is to get file infos, get the full path of a file where a object is saved in the filesystem, creating a new tour file, creating a new log file, getting all files, searching in the filesystem for a specific file, getting a file text and creating a image for a tour via integration of the mapquest API.
