Detailed Description

MainViewModel class which serves as the ViewModel for the MainWindow. It allows for the creation, deletion and modification of tours and logs for the tours. Furthermore it allows for import and export of tours and logs from and into a file. And lets the user generate a report of a tour with its corresponding logs.
